## 주요 업데이트 사항

<prose.h3>
  <PaymentV1 />
  ✔️ (신) 나이스페이먼츠 간편결제 허브형으로 알리페이(alipay)를 사용할 수 있습니다.
</prose.h3>

기존에는 국내 간편결제 수단만 지원 가능하였으나 이제 알리페이도 사용이 가능합니다. 알리페이 사용을 원하시는 경우 나이스페이먼츠와 계약한 이후 사용이 가능하오니 포트원으로 문의해 주시기 바랍니다.

(신)나이스페이먼츠 연동에 대해 자세히 알고 싶다면 [→ (신)나이스페이먼츠 연동가이드](/docs/ko/pg/payment-gateway/nice-v2/readme?v=v1) 에서 확인하실 수 있습니다.

<prose.h3>
  <PaymentV1 />
  ✔️ 결제 취소 시 발송되는 웹훅에 취소내역 아이디 (`cancellation_id`) 정보가 포함되었습니다.
</prose.h3>

기존에는 웹훅 발송 시 결제번호(`imp_uid`), 주문번호(`merchant_uid`), 결제 결과(`status`) 정보만 전달하였으나, 이제 취소내역 아이디(`cancellation_id`)도 포함하여 전달됩니다.
